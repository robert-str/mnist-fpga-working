# Notebooks - Two-Hidden-Layer Network

Jupyter notebooks for training the multi-layer neural network.

## Files

| Notebook | Description |
|----------|-------------|
| `siec_2_ukryte.ipynb` | Training notebook for 2-hidden-layer network |
| `siec_INT8.ipynb` | INT8 quantization experiments and export |

## Purpose

- Train PyTorch neural network on MNIST
- Apply post-training quantization (PTQ)
- Evaluate quantized model accuracy
- Export quantized weights for FPGA
